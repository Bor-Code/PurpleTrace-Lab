# Safety Scope

PurpleTrace Lab is a defensive security education project.

It is designed to demonstrate endpoint telemetry analysis, detection engineering, alert enrichment, and reporting.

---

## Allowed Scope

The project is limited to:

* Loading static sample telemetry
* Reading Windows Sysmon process creation events
* Applying detection rules
* Generating alerts
* Mapping alerts to MITRE ATT&CK
* Exporting reports
* Testing defensive detection logic

---

## Not Included

This project does not include:

* Malware
* Exploit code
* Shellcode
* Process injection
* Credential theft
* Persistence techniques
* Evasion logic
* AMSI bypasses
* Unauthorized scanning
* Unauthorized access
* Destructive behavior

---

## Purpose

The purpose of this project is to show safe defensive engineering skills.

It is suitable for:

* Portfolio demonstration
* Detection engineering practice
* Purple Team learning
* Windows telemetry analysis practice
* Rule-writing practice

---

## Responsible Use

Use this project only in environments you own or have permission to analyze.

The included sample files are safe static examples.
