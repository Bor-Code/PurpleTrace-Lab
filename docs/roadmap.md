# Roadmap

PurpleTrace Lab is currently focused on being a clean and explainable defensive portfolio project.

---

## Completed

* CLI-based detection pipeline
* JSON rule loading
* Rule validation
* Rule catalog listing
* Sample event loading
* Batch event loading
* JSONL / NDJSON event loading
* Sysmon process creation event collection
* Severity filtering
* MITRE technique filtering
* Rule ID filtering
* Rule tag filtering
* Alert evidence enrichment
* JSON export
* Markdown export
* CSV export
* HTML export
* Summary JSON export
* Unit tests
* GitHub Actions build workflow

---

## Planned Improvements

* Add screenshots to README
* Add rule coverage documentation
* Add more safe sample telemetry
* Add more defensive detection rules
* Prepare v0.1.0 release notes
* Improve HTML report layout
* Add example investigation workflow

---

## Not Planned

The following are intentionally out of scope:

* Offensive payloads
* Malware execution
* Exploit development
* Credential theft simulation
* Evasion or bypass techniques
* Process injection
* Persistence mechanisms
